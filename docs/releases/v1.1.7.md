# Hacker Bob v1.1.7 Release Notes

## Highlights

`v1.1.7` adds operator-controlled egress profiles for geofenced or region-sensitive targets. Existing Bob runs are unchanged unless an operator explicitly selects a profile:

```text
/bob-evaluate target.com
```

still uses the direct `default` egress path. To use a configured profile:

```text
/bob-evaluate target.com --egress gr-residential
```

## Changes

- Added `.claude/bob/egress-profiles.example.json` and installer-created `.claude/bob/egress-profiles.json` with a required direct `default` profile.
- Added `/bob-egress` plus `.claude/hooks/bob-egress.js` for listing, adding, testing, enabling, disabling, and removing profiles without printing proxy URLs or credentials.
- Added `bounty_http_scan.egress_profile`, proxy-backed request support through `proxy-agent`, and validation for `http`, `https`, `socks5`, and `socks5h` proxy URLs.
- Added `egress_profile` and `egress_region` to HTTP scan responses, HTTP audit records, compact audit summaries, evaluator briefs, and pipeline analytics. Proxy URLs are never recorded.
- Added geofence/reachability warnings for repeated first-party timeout, connection reset, and `network_unreachable_target` clusters in audit, circuit-breaker, status, debug, and analytics surfaces.
- Updated `/bob-evaluate --egress <profile>` to pass the selected profile through AUTH scans and every evaluator, chain, verifier, and evidence prompt. Profile switching remains explicit and operator-controlled.
- Updated installer, doctor, uninstall, dev-sync, package, and release checks so the new command, helper, example config, and runtime dependency are shipped.

This release also includes the pending no-findings workflow fixes from `main`: zero-reportable VERIFY results now close through SKIP grading and a no-findings report, evaluator prompts have clearer repeated `INTERNAL_ERROR` handoff rules, and force-merge reasons are recorded for wave settlement and analytics attribution.

## Install

```bash
npx -y hacker-bob-cc@1.1.7 install /path/to/your/project
```

After installing, fully restart Claude Code in that project so the new `/bob-egress` command and updated `/bob-evaluate` skill reload.
