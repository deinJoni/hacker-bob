# Hacker Bob v1.1.9 Release Notes

## Highlights

`v1.1.9` makes first-run onboarding even more direct for AI coding agents. If a user pastes the Hacker Bob repository URL into an agent while sitting in a normal Claude Code project directory, the agent should install Bob into that current directory instead of asking what to do next.

## Changes

- Updated README AI-agent instructions to treat a pasted repository URL as an install request.
- Made the default install command `npx -y hacker-bob-cc@latest install "$PWD"`.
- Kept the MCP verification steps immediately after install: load `./mcp/server.js` and run `claude mcp list`.
- Clarified that agents should ask for a target path only when the current directory is clearly unsafe or ambiguous, such as the Hacker Bob source checkout itself, a home directory, or a root directory.
- Moved source-clone installation into a fallback path for npm outages or explicit source-install requests.

## Install

```bash
npx -y hacker-bob-cc@1.1.9 install /path/to/your/project
```

After installing, fully restart Claude Code in that project and verify `claude mcp list` shows `bountyagent` connected.
